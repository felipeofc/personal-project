Java

O java é uma linguagem de programação de computadores de alto nível, bastante utilizada no cotidiano. Além de ser muito popular, o java é gratuito e multiplataforma, ou seja, é executável em todas as plataformas. Diverssas aplicações que conhecemos podem utilizar Java, desde as simples quanto as mais complexas. Como para um simples gerenciamento de dados, quanto no grande mercado de TI onde existem funcionalidades bastante complexas. Algumas qualidades do Java são bem notórias e algumas até singulares comparadas a outras linguagens de programação, tais delas são a simplicidade, Orientada a objetos, quase tudo no java é considerado um "objeto", menos os tipos primitivos, tendo comportamentos e estados, robusto, pois eele verifica todos os erros durante a compilação e execução, entre outras. A composição do java é basicamente o JVM (Java Virtual Machine), uma máquina abstrata, e o JRE (Java Runtime Environment) que é basicamente um pacote de software que tem as funções básicas para executar o programa incluindo toda a JVM. Não poderíamos esquecer do JDK (Java Development Kit), é a ferramenta necessária para você desenvolvedor java, ele contem toda a JRE e todos os interpretadores, compiladores e gerador de documentos para o desenvolvimento Java no geral.

Tipos primitivos

Como vimos acima, o Java é orientado a objetos, no entanto não é totalmente. Nele existem os tipos primitivos que são dados que servem para representar valores simples, e não classes. Usamos muito esses tipos primitivos em declarações de variáveis, onde de colocamos o nome da mesma e o seu tipo. Mas existem casos especiais que precisamos utilzar tipos diferentes de variáveis, onde queremos referênciar uma classe ou objeto, que são chamadas de variáveis de referência.

Os tipos primitivos do Java são: boolean, char, byte, short, long, int, float e double. Onde são separados por seus tipos, que são: lógicos, literais, inteiros e reais, com diferentes cadeias de bits.

Encapsulamento

O encapsulamento é uma das partes mais importantes da programação oriientada a objetos, é o processo no qual os programas são separados por partes, deixando essas partes mais separadas possíveis o progranma fica com uma melhor performance e organizado, deixando mais fácil de modificar e implementar outras aplicações ao mesmo, e também cabe citar que fica mais seguro, pois os funcionamentos dos nossos métodos ficam guardados, pois todas as classes ficam guardadas em diferentes pacotes, em diferentes hierarquias.

Herança

Como o próprio nome já diz, herança, é basicamente quando uma classe herda características de outra classe. Esse evento ocorre muito quando criamos uma classe Java a partir de outra. isso é muito útil para minimizar a escrita de códigos que vão ser utilizados em mais de uma classe em um mesmo projeto, pois esse processo faz a classe herdar todos os métodos, atributos, funções e contsrutores da outra classe já criada. Sendo assim, todo esse processo possui uma determinada hierarquia entre essas classes. As classes que herdam de outra são classificadas como classes filho ou subclasses, já as classes que estão sendo herdadas são denominadas classes pai, ou superclasses. No Java mesmo existe essa hierarquia de clases, pois toda classe existente no java herdam de Object que é a superclasse.

Polimorfismo

Difrente da herança, na qual as classes herdam apenas as características e métodos da classe pai, o Polimorfismo tem uma característica distinta. Essa técina faz com que os objetos possam se comportar de maneiras distintas, diferente do que foi herdado da superclasse. Essa implementação é feita a partir dos métodos, diferente da herança que é feita pela classe. Sendo assim, vemos que esses mecanismos de herança e polimorfismo são muito importantes na POO, tanto ajudam na reutilização do código quanto na implementação de novas ações no projeto.
